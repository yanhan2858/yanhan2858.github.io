团子翻译器配置千问API教学

这是继上篇文章用Deepseek API配置团子翻译器玩莉莉天使后的文章。 最近发现了阿里云的百炼控制台可以白嫖千问还有其他的模型于是就有了这期的素材。 当然，阅读时还请注意文章的时效性。因为白嫖活动可能随时过期。

好吧！长话短说，直接步入正题。如果你还不清楚团子翻译器是什么还请阅读这篇团子翻译器的 入门文章。

首先打开我们的浏览器点击下面的链接进入百炼控制台。我们先注册一个账号。
百炼控制台（点我！）

我们点击右上角的登录，先注册阿里云百炼控制台。如果你之前有阿里云账号那么你就可以直接拿来登录了。

博主这边有帐号了，所以我直接登录了哈！

首次登录平台会赠送3个月的Token额度，大概有200个可用模型。每个模型100万免费Token可用。

我们来到模型页面，然后还是跟之前一样。先创建一个API Key。

点击页面左下角的API Key

创建API Key，往里面填入基本信息。

复制API Key，当然你也可以下载保存。

接下来我们打开团子翻译器，来到在线AI翻译设置页面。这里我直接放老图了。

填写样式如下图所示。

这里特别强调一下，接口的请求地址不能用下图中的这个！

https://dashscope.aliyuncs.com/compatible-mode/v1

如果你用了上面这个接口地址调用时则团子翻译器会出现以下错误。如下图。

得用这个OPENAI兼容的接口请求地址。请使用下面这个接口地址。

https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions

这个就能跑通了。顺便一提，咱之前用的提示词是下面这个。



由于平台存在敏感词检测，这个提示词会触发内容审查随后阻断导致无法调用API。 如下图。所以你没办法用百炼控制台里边的模型翻译R18内容。 不管你调用的是什么模型。所以请换成这个保守一点的提示词。



我们不需要删除原有的提示词，直接在原有提示词的基础上粘贴上新的提示词即可。如下图所示。

粘贴完毕后，我们点击测试确保API调用正常。

前面提到虽然有200个可用模型可用调用。每个模型100万Token额度。

但是不是所有模型都可以在团子翻译器中完成翻译任务。所以需要你自行测试可用的模型。 目前Qwen3-MAX， 以及Qwen3-Vl-Plus均可用。 图像生成模型不能用于翻译。

我们可以来到模型用量界面批量开启模型免费额度用完即停。这样可以避免不必要的计费。因为即使额度用完了。API依然可以调用，这时候要花点小钱。

全部操作完毕后我们就可以上游戏测试了。这里我以花骑士为例。

可以看到千问的翻译效果还是很好的！当然翻译过程中你可能会遇到以下问题。

  1. 翻译提示返回的不是JOSN格式怎么办？

    2. 可以重新点击翻译，或者调整翻译框的识别区域。有时候本地OCR识别不那么准确。

  2. 翻译效果很差怎么办？

    3. 可以多翻译几次，或是编辑原文删除OCR识别的不正确的符号。尝试将原文换行进行翻译。或是逐行翻译原文。必要时你可以手动输入日文原文。

  3. 翻译时间长了模型一直返回的结果是日文怎么办？

    4. 尝试换一个句子翻译，打乱原有的上下文。这问题常有，原因不明。 强大的模型没这个问题。不过，换ChatGPT，Grok可以改善。 若还是一直返回日文可以调整提示词顺序。重启软件一般可解决。

  2. 创建API Key
  3. 配置API Key
  4. 开心玩耍
